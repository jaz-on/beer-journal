### backlog post‑mvp (ne pas générer dans la v1)

Ce document regroupe toutes les idées et fonctionnalités prévues après le MVP, afin d’éviter qu’elles soient mélangées à la documentation de la première version et générées par l’agent IA.

#### Ideas for future versions
- Statistics
  - Advanced statistics, charts (Chart.js), dashboard widget
- Export/Import
  - CSV Export, JSON Export, CSV/JSON Import (générique)
- Search and Filters
  - AJAX search, filtres avancés, persistance des filtres
- Maps & Location
  - Carte interactive, suivi de lieu (géocodage, lat/lng)
- Personal Features
  - Wishlist, Private Notes, Cellar Management
- Platform Integration
  - BeerAdvocate, RateBeer, Universal Import
- Taxonomie `beer_menu` (collections éditoriales) – exposée via blocks + filtres
